# 🐍 Python-AIML-Tasks

**Python for AI/ML Engineering** — Step-by-step tasks to go from Data Scientist to AI/ML Engineer.

## 🎯 Goal

Learn production-ready Python skills through small, incremental tasks — each building on the last.  
Focus: **Engineering side of Python** (not web dev).

## 📦 What's Covered

| Category | Topics |
|----------|--------|
| Core Python | OOP, type hints, dataclasses, decorators, logging |
| Dev Workflow | Git, GitHub, project structure, packaging |
| Data Processing | pandas, NumPy, polars |
| Backend & APIs | FastAPI, Pydantic, uvicorn |
| Databases | SQLite, SQLAlchemy, CRUD |
| MLOps | Experiment tracking, model registry, MLflow |
| Automation | argparse, click, scheduling |
| DevOps | Docker, CI/CD, env management |
| Performance | async/await, threading, profiling |
| Testing | pytest, mocking, linting |
